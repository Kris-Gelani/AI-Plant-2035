# AI-Plant-2035
AI Plant 2035

An Industrial AI Digital Twin platform developed to simulate and monitor a Green Methanol Production Plant.

Features:
• Process Simulation
• Water Electrolysis
• CO₂ Utilization
• Methanol Production
• Digital Twin Dashboard
• AI Copilot
• Historian Database
• Carbon Analytics
• ESG Monitoring
• Executive KPI Dashboard

Tools:
Python, DWSIM, SQLite, AI Analytics
